Ce fichier README contient des informations sur le site vitrine Findly, conçu pour aider les papas ou futurs papas à devenir plus responsables. Le site est développé en utilisant HTML, CSS et des polices personnalisées. Son objectif est de fournir des ressources précieuses, des informations et une plateforme communautaire pour les papas afin qu'ils puissent apprendre et évoluer dans leur parcours de parentalité.

Technologies Utilisées
HTML
CSS

Remarque
Ce site vitrine est un projet fictif créé à des fins d'apprentissage et de démonstration avec la formation MGK. Les informations et les ressources fournies sur le site sont également fictives.
